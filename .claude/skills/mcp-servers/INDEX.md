## Usage

These MCPs are already integrated with Claude Code. Tools are available via the `mcp__` prefix:
- Playwright: `mcp__playwright__*`
- Figma Desktop: `mcp__figma-desktop__*`

## Server Details

### Playwright
Browser automation for testing, screenshots, snapshots, and interaction with web applications.

### Figma Desktop
Extract design context, screenshots, metadata, and code from Figma designs for implementation.
