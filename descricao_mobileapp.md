# Tela: Login

- Login e senha basicos
- Botão entrar, vai para `Configurando sessão` 

# Tela: Configurando sessão

- Botão Logout, vai para `Tela: Login` 
- Seleção de voluntário
- Seleção de SNOMED estrutura do corpo sendo medida
- Seleção de Snomed Lateralidade
- Multi-Seleção de Snomed Topologia
- Seleção do dispositivo
- Seleção do sensor
- Conectar com dispositivo bluetooth
- Botão Iniciar sessão, vai para `Tela: Iniciando Gravação` 

# Tela: Iniciando Gravação

- Botão adicionar anotações médicas a Sessão, vai para `Tela: Anotações médicas`
- Botão iniciar uma gravação, vai para `Tela: Capturando` 
- Botão Finalizar sessão, vai para `Tela: Configurando sessão` 

# Tela: Capturando

- Grafico do sEMG atualizado a cada 0.5s
- Botão parar gravação, vai para `Tela: Iniciando Gravação`

# Tela: Enviando informações

- Exibe loading com mensagem: "Persistindo informações
- Vai para `Tela: Iniciando Gravação` 

# Tela: Anotações médicas

- Lista de anotações, botão excluir e visualizar
- Botão Adicionar Nova anotação, vai para `Tela: Nova Anotação médicas`
- Botão voltar, vai para `Tela: Iniciando Gravação` 

# Tela: Nova Anotação médicas

- Caixa de texto
- Botão Salvar, vai para `Tela: Anotações médicas`  
- Botão Cancelar, vai para `Tela: Anotações médicas`
