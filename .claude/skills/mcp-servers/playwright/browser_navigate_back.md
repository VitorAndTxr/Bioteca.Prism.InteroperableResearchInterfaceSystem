# browser_navigate_back

Go back to the previous page.

**Access**: `mcp__playwright__browser_navigate_back`

## Parameters

None

## Returns

Success status

## Errors

- No previous page in history
- Browser closed

## Example

```typescript
// Navigate back
mcp__playwright__browser_navigate_back({})
```

## Common Use Cases

- Testing browser navigation
- Returning to previous state
- Testing back button functionality
